# Asset Licenses And Provenance

Record attribution and provenance for any committed non-original asset.

Use one entry per asset or asset set:

```text
Asset path:
Source path:
Creator or tool:
License or usage basis:
Source URL:
Date added:
Prompt summary, if generated:
Edited after generation:
Intended use: prototype / production candidate
Restrictions:
```

If provenance is unclear, do not commit the asset.
