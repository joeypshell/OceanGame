# Asset Sources

Store editable source assets here before exporting runtime files.

## Subfolders

- `concepts/`: mood studies, generated concepts, sketches, and exploratory references.
- `sprites/`: editable source files for gameplay sprites.
- `vectors/`: vector source files for icons, markers, silhouettes, and UI-readable shapes.
- `ui/`: UI-specific source art.

## Rules

- Keep names role-first, such as `resource_glow_plankton_sprite_v1.png` or `lock_pressure_gate_v1.svg`.
- Record generated-image prompt summaries and provenance in `assets/licenses/`.
- Do not commit unclear-provenance assets.
- Do not use source art to own gameplay state; source art skins or replaces readable prototype visuals.
