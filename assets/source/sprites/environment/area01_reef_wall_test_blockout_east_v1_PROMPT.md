# Area 01 Reef Wall Test - Blockout East v1

Generated as a one-wall art-direction test for replacing a flat Area 01 black-box wall placeholder.

## Intended Test Target

- Source-map terrain id: `blockout_east_reef_mass`
- GitHub issue: `#699`

## Prompt

Create a single horizontal reef/cave wall platform sprite to replace a flat black blockout wall placeholder.

The asset should be a long modular underwater rock wall segment with a dark cave-like underside and readable top/edge silhouette. It should look like layered blue-black stone plates with subtle teal rim highlights, small coral/kelp accents, and a few dim bioluminescent details. The top edge should be clearly solid terrain but not overly bright.

Style should be polished stylized 2D game art, painterly/pixel-adjacent, readable at normal side-view play scale, inspired by underwater exploration games without copying a specific asset.

Generated on a flat `#ff00ff` chroma-key background and converted to transparent PNG with local chroma-key removal.
