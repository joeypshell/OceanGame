# Assets

This folder holds committed art source conventions for OceanGame.

Use this tree for source art, runtime-ready exported assets, and license/provenance notes. Do not put Godot editor caches, imported `.import` metadata, local screenshots, or platform builds here.

## Folders

- `source/`: editable source files, prompts, references, and concept notes.
- `exports/`: committed runtime-ready asset files that the Godot project may reference.
- `licenses/`: attribution, provenance, and license notes for non-original assets.

See `docs/planning/ASSET_SOURCE_PIPELINE_V1.md` for the planning rationale.
