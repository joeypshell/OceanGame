# Pressure Wreck/Cache Backup Art-Pass Constraints - 2026-06-26

Status: planning artifact for issue #271.

## Purpose

Define what the pressure wreck/cache backup art pass may do if it becomes the next source-art replacement after the surface moonpool/waterline priority.

This is a constraint plan, not an implementation issue.

## Current Role Ownership

| Role | Owner | Preserve |
| --- | --- | --- |
| Pressure access | Gate shimmer, bars, lock/open badge, pressure route copy | The gate decides blocked/open readability. |
| Outside clue | Outside wreck scan target | The outside scan points toward pressure-route understanding, not a reward or map marker. |
| Cache payoff | Wreck Signal Cache visual and scan behavior | The cache is a local mystery payoff inside the wreck stack. |
| Echo Lens feedback | Temporary local pulse around the cache | Echo Lens gives broad local echo feedback, not exact navigation. |
| Predator danger | Gulper lane, warning ribs/current, predator sprite/state | Predator remains avoidable route-timing pressure, not combat or loot. |
| Route memory | Result/ready copy and discovered-place memory | The player remembers the place without a quest checklist. |

## Allowed Backup Art Pass

If this backup pass is selected, it may:

- add one source/export/provenance asset or a small matched set for the pressure wreck/cache environment shell,
- replace or tune wreck hull, broken bow, interior shadow, gate surround, cache alcove, and pressure-dark atmosphere visuals,
- subdue redundant fallback geometry only after the source-art read is clearer at normal scale,
- tune alpha, depth ordering, or local contrast to reduce predator-lane and reward-cluster overlap,
- keep scan markers, pressure badges, and Echo Lens pulse readable as separate concepts,
- add a normal-scale screenshot review for locked, open, cache, Echo Lens pulse, and predator-overlap states.

## Preserve

- Existing pressure gate behavior and `Pressure Seal I` ownership.
- Existing outside scan behavior and pressure-route clue language.
- Existing Wreck Signal Cache scan and repeat-scan behavior.
- Existing Echo Lens pulse behavior and no-locator copy rules.
- Existing predator route behavior, warning, chase, contact, and decoy interactions.
- Existing player movement, oxygen, cargo, extraction, save/load, and progression behavior.
- `Wreck Echo Descent` as planning-only.

## Avoid

- Implementing `Wreck Echo Descent`.
- Adding a new route, region, biome, cache reward, predator behavior, resource, upgrade, or procedural generator.
- Adding a minimap, route graph, exact scanner locator, field-guide UI, quest log, objective checklist, or persistent marker.
- Brightening pressure gate, cache, Echo Lens pulse, reward lure, and predator lane at the same time.
- Turning predator danger into combat, harvesting, health, or loot.
- Moving gameplay nodes into art-only ownership.

## Visual Constraints

The backup pass should reduce shape pileup, not add another bright object to the cluster.

Preferred direction:

- bigger, calmer wreck silhouette,
- fewer overlapping placeholder polygons,
- clearer gate frame,
- cache alcove separated by local light and shadow,
- subdued pressure-dark atmosphere behind the route,
- predator lane visually below/nearby but not fused with the gate/cache.

Avoid:

- icon-like cache/gate treatment,
- global beacon or vertical objective beam,
- UI arrow or distant marker,
- text-heavy solution,
- brighter reward lure than the pressure access read.

## Acceptance For A Future Implementation Issue

A future implementation issue should require:

- source, export, and provenance files,
- scene wiring under existing pressure wreck/cache visual ownership,
- normal-scale evidence for locked gate, outside scan, open route, cache, Echo Lens pulse, and predator overlap,
- no gameplay behavior changes,
- no Wreck Echo Descent implementation,
- Godot headless launch,
- logic tests if scene contracts or route behavior change,
- `git diff --check`.

## Decision

Use this as the backup art-pass contract if pressure wreck/cache becomes the next source-art target.

The pass is allowed to clarify the existing lower-route scene. It is not allowed to expand the game.
