# Next Source Art After Evidence Hardening - 2026-06-26

Status: selected narrow target.

Issue: #395.

## Decision

Select the lower-route `DeepRewardLure` / reward-cue cluster as the next narrow source-art readability target.

The goal is not a full lower-route repaint. The goal is to make the optional deep reward read less like the pressure/cache/Wreck Echo signal family and less like predator danger, using one authored source-art pass plus conservative runtime wiring.

## Why This Target

The hardened lower-route evidence shows:

- oxygen/base safety remains readable,
- predator danger remains readable,
- pressure/cache opportunity remains readable enough,
- Wreck Echo remains an optional downstream research clue,
- the remaining visual risk is bright yellow/green reward, cache, and route cue lights clustering in the same lower-right band.

The deep reward lure is the best next target because it can reduce that competition without changing route layout, scanner rules, pressure access, Wreck Echo state, predator behavior, HUD layout, cargo rules, or progression.

## Target Scope For The Next Implementation

The next issue may:

- add one source/export/provenance asset for the deep reward lure or reward-cue shell,
- wire that asset into the existing `DeepRewardLure` visual slot or equivalent child visual,
- reduce or subordinate fallback glow geometry only as needed for readability,
- preserve all gameplay nodes, collision, resource pickup behavior, scan behavior, predator behavior, pressure rules, and Wreck Echo gating,
- verify with the existing Playwright lower-route capture.

The next issue must not:

- add a new resource type, inventory grid, tech tree, field guide, minimap, route graph, quest marker, or objective checklist,
- expand Wreck Echo,
- add combat/capture/harvest loops,
- move the lower-route layout,
- implement controller or touch controls.

## Visual Direction

The reward lure should read as optional material opportunity:

- warmer amber/gold or mineral-container language is acceptable,
- avoid green/cyan signal language that competes with cache/Wreck Echo,
- avoid red silhouettes that compete with predator danger,
- keep the silhouette compact enough that oxygen/base and predator danger stay louder.

## Acceptance Criteria For The Next Implementation

- `lower-route-pressure-gate.png` still shows oxygen/base, predator danger, pressure/cache, and reward as separate meanings.
- The reward lure is visually quieter than low/critical oxygen and predator danger.
- The reward lure is more distinct from Wreck Echo and cache cues than the current placeholder cluster.
- No behavior or progression changes occur.
