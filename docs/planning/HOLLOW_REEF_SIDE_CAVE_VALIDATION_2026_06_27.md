3. Is there a clear reason to enter and return safely?

Pass. The `Hollow Reef cave reading` is a visible one-run payoff, dims after recovery, is duplicate-safe, and adds compact extraction memory if the player returns. It does not add cargo, a quest checklist, durable route state, or a field-guide objective.

4. Does the branch promise future depth without opening a broad cave system?

Pass. `DeeperReefPromise` is a visible closed mouth with seal lip, pressure wash, glint, and `DEEP REEF CLOSED` copy. It has no interaction, collision, pressure boundary, resource pickup, interior lane, unlock behavior, or durable save state.

5. Is no-minimap viability still credible?

Pass with watchlist. Hollow Reef works with current landmarks, return-current shape language, HUD base direction, result memory, and staged screenshot evidence. The watchlist is that normal play still needs better planning and result copy across the full lower-route chain so players know why to aim for Blackwater, Dusk, and Hollow Reef on a given day.
