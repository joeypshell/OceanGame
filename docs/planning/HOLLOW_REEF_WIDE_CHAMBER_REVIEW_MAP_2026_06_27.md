## Why Now

This is the right point to draw a map. The project has moved from a single cave mouth into a short interior lane and the first wide chamber. Before adding named landmarks, payoff content, a creature route, salvage pockets, and route-goal copy, the area needs a shared visual plan so the next issues add meaningful playable space instead of another dense cue pile.

The v2 map uses a Dave-the-Diver-style exploration-atlas format: black background, dotted depth lines, chunky rock clusters, icon callouts, creature placement labels, and a legend. It is better for evaluating level shape and content density than the cleaner v1 route diagram.

## What The Map Represents

- Dusk Trench as the existing lower-route approach.
- Glass Kelp Ledge as an already implemented side pocket.
- Hollow Reef as the current cave branch and reading payoff.
- Hollow Reef Interior Lane as the bridge into the new chamber.
- Wide Reef Chamber as the current #574/#575 route-breadth target.
- Future slots for chamber payoff, creature route, and deeper promise.

## Design Guardrails

- This is a planning/reference map, not an in-game minimap commitment.
- Keep normal play side-view and landmark-driven.
- Use the map to decide content placement before creating more local polish.
- Do not add exact route graph UI, checklist objectives, field-guide quest state, or procedural cave generation from this artifact alone.
